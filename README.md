## ☁️ AWS Cloud Projects

### Project 1 — S3 Static Website + CloudFront CDN

A production-grade static website hosted on S3 with CloudFront CDN, HTTPS, and Origin Access Control (OAC) for secure private bucket access.

**Services Used:** S3, CloudFront, ACM, OAC

**Key Highlights:**
- Static website hosting on S3
- CloudFront CDN with 400+ edge locations
- HTTPS enabled with default SSL certificate
- S3 locked down — only CloudFront can access via OAC
- Cache invalidation for instant updates

### Project 2 — EC2 Web Server with Nginx

A fully configured web server running on AWS EC2 with Ubuntu 22.04 and Nginx. Includes security group configuration, UFW firewall, and SSH key-based access.

**Services Used:** EC2, Security Groups, Key Pairs

**Key Highlights:**
- EC2 instance with Ubuntu 22.04 LTS
- Nginx installed and configured as web server
- Security Group: SSH (My IP) + HTTP (Anywhere)
- UFW firewall as second security layer
- Custom website deployed to /var/www/html/

### Project 3 — DevOps Auto Deploy Pipeline

A CI/CD-style deployment pipeline where code pushed to GitHub is automatically deployed to an EC2 server every 5 minutes using Bash scripting and Cron.

**Services Used:** EC2, Nginx, Git, GitHub, Bash, Cron

**Key Highlights:**
- GitHub as source control
- Bash deploy script automates pull + copy + restart
- Cron job runs deployment every 5 minutes
- Zero manual intervention after git push
- Deployment logging for history tracking
